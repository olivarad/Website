Macro Keyboard

Project Repository Main

Welcome to the dedicated page for my Macro Keyboard Project. This device features fully customizable keys, empowering users to program each key for a multitude of functions, from launching applications to facilitating actions like copying and pasting.

Future Changes

Upon designing and receiving the initial version of the PCBs, I realized that the extensive number of keys could result in confusion and high production costs. Consequently, I have chosen to streamline the design by reducing the number of keys and incorporating rotary encoders, enabling users to switch between keyboard modes in an intuitive manner. Additionally, I've decided to integrate a small liquid crystal display (LCD) into the device to display the mapping of each key. This approach not only reduces production costs but also enhances user experience by providing a clear reference for programmed key functions.
